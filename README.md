J_GATE v1.0
Admin Panel Finder & Login Tool
Installation
pip install requests colorama
Usage
python j_gate.py
Features
Find Admin Panels - Scan and locate login/admin pages
Brute Force Login - Test common credentials automatically
Custom Credentials - Test your own username/password combinations
Fast & Reliable - Multi-path scanning with HTTP status detection
Menu Options
[1] Find Admin Panels      - Scan for login pages
[2] Brute Force Login      - Try common credentials
[3] Custom Credentials     - Test custom login
[4] Exit                   - Quit tool
Supported Paths
login, admin, wp-admin, cpanel, phpmyadmin
And 30+ more common paths
